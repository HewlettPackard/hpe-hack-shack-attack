# discover-2019-game
Hackshack game for Discover 2019 event

## Getting Started
Install dependencies individually for frontend, game-client, and server folders

```
cd frontend
yarn
```
```
cd game-client
yarn
```
```
cd server
yarn
```


## To run

frontend
```
cd frontend
yarn start
```
game-client
```
cd game-client
yarn start
```
server must have MongoDB running prior
```
cd server
yarn-watch
```
